# Sales-Performance-Dashboard

## 📌 Project Overview
This project presents an interactive Sales Performance Dashboard created to analyze sales data and identify key business insights. The dashboard provides a comprehensive view of sales across different regions, categories, and customer segments.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify top-performing categories and products  
- Understand customer purchasing behavior  
- Compare sales across regions and segments  

---

## 🛠️ Tools & Technologies
- Microsoft Power BI / Microsoft Excel  

---

## 📊 Dataset
The dataset used for this project was sourced from Kaggle and contains sales-related information such as product categories, customer segments, and regional performance.

---

## 📈 Dashboard Insights
- Technology category generated the highest sales  
- Phones and machines are top-performing sub-categories  
- Certain states contribute significantly to total sales  
- Top customers account for a major share of revenue
